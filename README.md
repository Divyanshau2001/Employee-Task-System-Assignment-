# Employee-Task-System-Assignment-
Employee task management system, where admins can add tasks, employees can log in an update the status of their tasks.
Setup -> 

1) Database
- Open mysql db and create DB schema as added in dbschema folder in code
- In Backend/src/config/db.js Add your password and username for DB

2) For running backend APIs
   - cd backend (Go to backend folder)
   - npm install
   - node src/index.js

3) For running Frontend
   - cd frontend (Go to frontend folder)
   - npm install
   - npm start
